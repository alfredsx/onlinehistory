Description

The module can be used to display users who are currently online.

install / uninstall

Installation is done as usual, following the standard installation process - Unpack the module into the ../modules directory. Install the module via Admin -> System Module -> Module.

For detailed instructions on installing modules, see the XOOPS Operations Manual
